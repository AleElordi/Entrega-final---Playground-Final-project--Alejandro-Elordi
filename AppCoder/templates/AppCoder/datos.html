Consigna

En forma individual, crearás una aplicación web estilo blog programada en Python en Django. Esta web tendrá admin, perfiles, registro, páginas y formularios.

  • Acceso visible a la vista de "Acerca de mí" en about/ = "Datos".
  • Acceso visible a la vista de blogs en pages/ = "Productos".
  • Pantalla de páginas con opción “Leer más” para navegar al detalle (pages/).
  • Si no existe ninguna página mostrar un "No hay páginas aún".
  • Solo usuarios logueados pueden editar o borrar páginas.

Requisitos base
  • Entrega individual en GitHub
  • Readme como la entrega 3
  • Video de máximo 10 min mostrando la página y funcionalidades
  • No agregar la Base de datos (db.sqlite3) en la entrega. Debe estar en el .gitignore
  • Uso de herencia de templates y NavBar en el template base
  • Existencia de .gitignore con pycache, db.sqlite3, media
  • Archivo requirements.txt actualizado
  • Adaptar templates y vistas para manejo de imágenes en formularios
  • Uso de mínimo 2 clases basadas en vista (CBV)
  • Uso de mínimo un mixin en una CBV y un decorador en una view común
  • Vista de inicio/home
  • Vista "Acerca de mí"/"About"
  • Modelo principal con mínimo: 2 CharField, 1 texto enriquecido (ckeditor), 1 imagen, 1 fecha
  • Listado de objetos del modelo principal mostrando algunos datos
  • Mensaje si no hay objetos creados o no se encuentra ninguno en el buscador
  • Detalle, creación, edición y borrado de objetos desde el listado
  • Registrar todos los modelos en el admin
  • App para manejo de usuarios/autenticación (accounts, cuentas, etc)
  • Vistas para login, logout y registro (username, email, password)
  • Vista de perfil mostrando: nombre, apellido, email, avatar, biografía/link/fecha de cumpleaños/etc.
  • Acceso a edición de perfil y cambio de password
  • App de mensajería para comunicación entre usuarios
